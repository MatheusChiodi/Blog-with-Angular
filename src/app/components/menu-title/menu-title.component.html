BLOG ANGULAR

Este projeto foi utilizado como forma de aprendizado da linguagem Angular.

O estudo de Angular foi realizado através da plataforma DIO, com foco em práticas modernas de desenvolvimento front-end.

Durante todo o desenvolvimento, o projeto foi executado em ambiente Docker, garantindo portabilidade e facilidade de configuração.

Tecnologias utilizadas:
Angular (framework principal), TypeScript (linguagem base), CSS3 (estilização), HTML5 (estrutura), Docker (containerização), Node.js (ambiente de execução), NPM (gerenciador de pacotes). Tailwind CSS (Biblioteca de estilização).
